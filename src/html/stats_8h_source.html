26 /* Statistics facilities
27  * Author: Daniel Sanchez
28  * Date: Aug 2010
29  *
30  * There are four basic types of stats:
31  * - Counter: A plain single counter.
32  * - VectorCounter: A fixed-size vector of logically related counters. Each
33  * vector element may be unnamed or named (useful when enum-indexed vectors).
34  * - Histogram: A GEMS-style histogram, intended to profile a distribution.
35  * It has a fixed amount of buckets, and buckets are resized as samples
36  * are added, making profiling increasingly coarser but keeping storage
37  * space constant. Unlike GEMS-style stats, though, at some configurable
38  * point part of the array starts growing logarithmically, to capture
39  * outliers without hurting accuracy of most samples.
40  * - ProxyStat takes a function pointer uint64_t(*)(void) at initialization,
41  * and calls it to get its value. It is used for cases where a stat can't
42  * be stored as a counter (e.g. aggregates, RDTSC, performance counters,...)
43  * or where we have values we want to output just as stats, but would not
44  * like to treat as raw counters because e.g. they have a different type,
45  * or for efficiency reasons (e.g. the per-thread phase cycles count is
46  * updated on every BBL, and may be an uint32_t)
47  *
48  * Groups of stats are contained in aggregates (AggregateStat), representing
49  * a collection of stats. At initialization time, all stats are registered
50  * with an aggregate, forming a tree of stats. After all stats are
51  * initialized, the tree of stats is made immutable; no new stats can be
52  * created and output at runtime.
53  *
54  * These facilities are created with three goals in mind:
55  * 1) Allow stats to be independent of stats output: Simulator code is only
56  * concerned with creating, naming, describing and updating a hierarchy of
57  * stats. We can then use a variety of *stats backends* to traverse and
58  * output the stats, either periodically or at specific events.
59  * 2) High-performance stats: Updating counters should be as fast as updating raw
60  * integers. Counters are objects though, so they entail some space overhead.
61  * 3) Allow fixed-size stats output: The stat types supported are all fixed-size,
62  * and stats cannot be created after initialization. This allows fixed-size records,
63  * making periodic stats much easier to parse and **iterate over** (e.g. we can
64  * parse 1% of the samples for a high-level graph without bringing the whole stats
65  * file from disk, then zoom in on a specific portion, etc.).
66  *
67  * This design was definitely influenced by the M5 stats facilities, however,
68  * it is significantly simpler, doesn't use templates or has formula support,
69  * and has an emphasis on fixed-size records for periodic stats.
70  */

132  //Returns true if it is a non-empty type, false otherwise. Empty types are culled by the parent.
133  bool makeImmutable() {
134  assert(_isMutable);
135  assert(_name != nullptr); //Should have been initialized
136  _isMutable = false;
137  g_vector<Stat*>::iterator it;
138  g_vector<Stat*> newChildren;
139  for (it = _children.begin(); it != _children.end(); it++) {
140  Stat* s = *it;
141  AggregateStat* as = dynamic_cast<AggregateStat*>(s);
142  if (as) {
143  bool emptyChild = as->makeImmutable();
144  if (!emptyChild) newChildren.push_back(s);
145  } else {
146  newChildren.push_back(s);
147  }
148  }
149  _children = newChildren;
150  return _children.size() == 0;
151  }

154  assert(_isMutable);

345  * Generic lambda stats
346  * If your stat depends on a formula, this lets you encode it compactly using C++11 lambdas
347  *
348  * Usage example:
349  * auto x = [this]() { return curCycle - haltedCycles; }; //declare the lambda function that computes the stat; note this is captured because these values are class members
350  * LambdaStat<decltype(x)>* cyclesStat = new LambdaStat<decltype(x)>(x); //instantiate the templated stat. Each lambda has a unique type, which you get with decltype
351  * cyclesStat->init("cycles", "Simulated cycles"); //etc. Use as an usual stat!
352  */
